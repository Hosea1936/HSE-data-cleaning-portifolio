# HSE incident Data Cleaning Using Power Query
## Project Overview
This project demonstrates end to end data cleaning showcasing how raw HSE incident data can be transformed into reliable, analysis-ready information using Power Query
### Data Quality Issues identified
-Missing values
-Duplicate records
-Inconsistent date formats
-Spelling errors
-Blank rows
#### Data cleaning steps
1.Removed dublicate records
2.Standardized date formats
3.Replaced inconsistent text values
4.Handled missing values
5.Removed unnecessary columns
##### Tools Used
- Microsoft excel.
- Power Query.
  ###### Project outcome
  The dataset was transformed into a clean and analysis-ready format suitable for reporting and dashboard development.
